# Scope (MVP)

This MVP focuses on platform foundations rather than individual application services.

In Scope:
- Terraform modules
- multi-env (`dev`, `prod`)
- VNet + basic subnetting
- App workload hosting (App Service or Functions)
- secrets management (Key Vault)
- logging and diagnostics (Log Analytics)
- RBAC baseline
- cost awareness (tags + budgets)
- GitHub Actions workflows for Terraform

Out of Scope:
- multi-subscription landing zones
- AKS / Kubernetes workloads
- enterprise policy frameworks
- private link networking
- disaster recovery
- multi-tenant models
- chargeback automation
